Checking the App Lab software

Log in Details

Some log in details have been sent to you.

  • A web address for the classroom.
  • Your log-in name.
  • Two 'secret words', which are your password.

You should have a 'dashboard' with various options.

Scroll down to 'Projects' and select 'App Lab'



Checking Drag & Drop

Now you are in App Lab, you should see a screen that looks like the picture below (it might look slightly different based on your computer).

  • Pretend phone on the left.
  • Menu of code blocks in the middle.
  • Blank space for code on the right.

Click the Design button on the top left.

This will replace the code blocks and window with some design options, like the below picture.


Drag and drop a Button from the middle menu onto the pretend phone on the left.

Then, on the right, click in its text property, and change it (to anything you like!).

Checking Code Blocks

If everything in the above section works, we can check if the code blocks work.

On the right hand section, where you just changed the text of the button, click EVENTS at the top.


Scroll down to find Insert and show code. Click this!


Your view will change again, and you should see some code on your screen.

On the right, click inside the purple code block that starts console.log. Check that you can change the text from "button1 clicked" to something else - keep the speech marks there.


Click Run on the bottom left.


Checking Clicks

When you click Run, the Run button will change to Reset.

Click on your Button in the middle of the pretend phone.

You should see some text show up in the Debug window on the bottom right.


Checking Text

Finally, we need to check that the text-coding works on your device.

On the top right, click the Show Text option. This will turn the coloured block code into text.

Click in the window on the right, with the code in it, and try typing there - type anything on any line, this is just to check that your keyboard input is recognised here.


If all the above checks worked, you will have no problems using App Lap on your device.
